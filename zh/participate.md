## 翻译

您可以使用以下命令来创建`nimf.pot`。

```
git clone https://github.com/cogniti/nimf.git
cd nimf
./autogen.sh
cd po
make nimf.pot
```

您可以将`nimf.pot`翻译成您的母语。
