## Ubuntu PPA

[https://launchpad.net/~hodong/+archive/ubuntu/nimf](https://launchpad.net/~hodong/+archive/ubuntu/nimf)

```
sudo add-apt-repository ppa:hodong/nimf
sudo apt update
sudo apt install nimf
im-config -n nimf
```
