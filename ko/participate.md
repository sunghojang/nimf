## 번역

다음 명령을 사용하여 `nimf.pot`를 만들 수 있습니다.


```
git clone https://github.com/cogniti/nimf.git
cd nimf
./autogen.sh
cd po
make nimf.pot
```

`nimf.pot`를 귀하가 사용하는 언어로 번역해도 됩니다.
